# Agentic Authorship Dashboard

Interaktiver Begleiter zum Buch **"Agentic Authorship"** — KI-gestuetzte Buchproduktion mit OpenClaw.

## Was ist das?

Dieses Dashboard ist sowohl Projektmanagement-Tool als auch Werkzeugkasten fuer Buchleser. Es besteht aus 12 eigenstaendigen Modulen, die unabhaengig voneinander funktionieren.

## Module

### Projekt & Verwaltung (M1–M6)
| Modul | Funktion |
|-------|----------|
| M01 | Projektuebersicht — Status auf einen Blick |
| M02 | Task-Tracking — Aufgaben verwalten |
| M03 | Sprint-Verwaltung — Woechentliche Planung |
| M04 | Content-Pipeline — Kapitel-Status verfolgen |
| M05 | Publishing / KDP — Amazon-Workflow |
| M06 | Wundertuete — SOUL.md & AGENTS.md Generator |

### Leser-Werkzeuge (M7–M12)
| Modul | Funktion |
|-------|----------|
| M07 | Workspace Wizard — 8 Fragen, alle Dateien generiert |
| M08 | Config Builder — openclaw.json visuell bauen |
| M09 | Cost Calculator — Provider vergleichen, Kosten kalkulieren |
| M10 | SOUL.md Gallery — 36+ Templates browsen und anpassen |
| M11 | CLI Referenz — 150+ Befehle durchsuchbar |
| M12 | Status Dashboard — Gateway live ueberwachen |

## Technik

- **Vanilla HTML/CSS/JS** — kein Build-Prozess, kein Framework
- **GitHub Pages** — jeder `git push` = automatisch live
- **localStorage** — Daten bleiben zwischen Sessions erhalten
- **Responsive** — funktioniert auf Desktop und iPhone

## Lokal starten

Einfach `index.html` im Browser oeffnen. Kein Server noetig.
